#Sistema de inscripción Academica con implementación de calidad de software
Este proyecto es un sistema de inscripción académica desarrollado con el objetivo de facilitar el proceso de registro de estudiantes en cursos y asignaturas. El sistema está diseñado para ser intuitivo y eficiente, permitiendo a los estudiantes inscribirse en sus cursos de manera rápida y sencilla.

## Características principales
- Interfaz de usuario amigable y fácil de navegar.
- Funcionalidad de búsqueda para encontrar cursos y asignaturas disponibles.
- Proceso de inscripción simplificado con confirmación de registro.
- Gestión de horarios para evitar conflictos entre cursos.

## Calidad de software
El sistema ha sido desarrollado siguiendo principios de calidad de software, incluyendo:
- Pruebas unitarias para garantizar la funcionalidad de cada componente.
- Revisión de código para mantener la legibilidad y mantenibilidad.
- Uso de patrones de diseño para mejorar la estructura del código.
- Documentación clara y detallada para facilitar la comprensión y el mantenimiento del sistema.
